Rolê Show: os melhores bares do centro de BH

Estamos cansados de escutar que BH é uma roça grande e que a nossa praia é o BAR.
Sejamos sinceros: acharam certo.
Para viver o melhor que o centro é capaz de oferecer, fizemos o guia Rolê Show: os melhores bares do centro de BH.
Completo, reunimos aqui apenas os (melhores) bares do centro: nada de Savassi, Lourdes e Buritis.
Aqui o negócio é cerveja barata, prosa boa e gente como a gente!

1. Bar Banzai

Barato, muita mesa, plano, agradabilíssimo. Tem até garotas de vez em quando. Certamente é o melhor bar de Belo Horizonte. Às vezes rola até barraco. A comida é arriscada. Fica lotado algumas vezes. Banheiro magnífico. Isso é o Banzai.
